A classic Sudoku game built in Java with a graphical interface.

## Download & Play
1. Go to the [Releases](https://github.com/jinhao0716/Sudoku/releases) page
2. Download `Sudoku-Windows.zip`
3. Unzip the folder
4. Run `Sudoku.exe`

## About
A classic 9x9 Sudoku game with a graphical interface built using Java Swing & AWT.
Features randomly generated Sudoku boards, different backgrounds during gameplay, and automatic tracking of cells that break Sudoku rules.

## Built With
- Java
- Java Swing
- Java AWT
